Soluções simples para problemas reais.

Transformar problemas em soluções simples é o que me move. Sou um desenvolvedor full stack. Transformo desafios em sistemas funcionais que fazem a diferença. Gosto de criar sistemas multifuncionais — verdadeiros “canivetes suíços” — que ajudam a otimizar o dia a dia de equipes e automatizar processos.

Tenho experiência com C#, Python, Flask, bancos de dados, automações, APIs e visualização de dados com Power BI. Me preocupo tanto com o funcionamento quanto com a usabilidade: gosto de tornar o front-end acessível e funcional.

Meus Projetos

AutoGest

Sistema web Flask com Bling API para gestão de autocenters, consulta de baterias, estoque e dashboard interativo.

BateriaFácil

Aplicativo desktop em C# com banco de dados embarcado (SQLite), desenvolvido para identificar baterias por modelo de carro. Acompanha instalador para facilitar a instalação e o uso no dia a dia comercial.

React Node Bateria

Projeto fullstack com API Node.js + Express gerando dados diretamente de um banco SQLite local. Frontend em React para consulta e recomendação rápida de baterias automotivas por veículo, marca e amperagem.

Backup Google DriveAPI

App desktop em C# que compacta e envia pastas locais ao Google Drive via API, com apenas um clique. Interface moderna e instalador incluso.

TkBackupDrive

Backup automatizado em Python com Tkinter, enviando pastas para o Google Drive com nomeação por data e hora.

Nfe Automation

Script Python com Selenium e PyAutoGUI para automação do cadastro de produtos e notas fiscais no ConnectUse.

Plennus

Aplicação desktop C# para importação e visualização de XMLs de NF-e, exibindo dados de emitente, produtos e impostos.

Estudo de Botões em C#

Projeto experimental para criação de botões estilizados e modernos em Windows Forms.

Combustível ou Gasolina?

Aplicativo simples em C# para comparar preço do etanol e gasolina e indicar o mais vantajoso com base na relação de custo-benefício.

Jogo da Velha 1.0 (XAML)

Projeto de estudo para interfaces em XAML com C#, focado em separar lógica e visual.

Resoluções do Beecrownd

Soluções em Python e Kotlin para problemas do Beecrowd (URI), reforçando lógica de programação e estrutura de código durante a graduação.
